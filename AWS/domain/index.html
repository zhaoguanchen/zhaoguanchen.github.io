AWS Route 53（域名）

Created2022-02-08|Updated2022-02-09|AWS
|Word count:161|Reading time:1min|Post View:

Domain

TransferDomainToAnotherAwsAccount

aws间域名迁移

采用aws cli的方式：

  • 原账号下发起域名迁移

参数为域名和账户ID

  • 返回操作ID和password（保存备用）
  • 新账号下域名接收

参数为域名和返回的password

  • 操作结束，此时域名迁移至新aws账号下。

https://stackoverflow.com/questions/62785304/route53-transfer-domain-to-another-aws-account
